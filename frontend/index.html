🚀 抢课程序运行指南

生成的项目包括两部分：grabber.py（核心运行引擎）与 config.json（包含登录凭证与抢课课程）。

1. 环境搭建

抢课脚本需要 Python 3.8+ 环境，并安装以下依赖：

pip install -r requirements.txt

或者手动安装核心依赖：

pip install requests ddddocr
注意：ddddocr 库在安装时可能需要编译器环境，如果在 M1/M2 Mac 上，需要通过 Rosetta 终端运行，或者配置对应的 ARM 库支持。

2. 配置文件配置

直接修改下载下来的 config.json，可以在其中更改密码、重试频率或课程任务，无需修改 Python 核心代码。

3. 启动运行

将 grabber.py 和 config.json 置于同一目录下，然后在终端中执行：

python grabber.py

程序会自动识别验证码进行登录，当登录失效或重试时会自动重试。成功登录后，所有配置的抢课任务将在独立的子线程中并发执行，直到所有课程抢课成功后，自动退出。
